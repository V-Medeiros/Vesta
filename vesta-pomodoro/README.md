# Vesta

Aplicacao React + TypeScript para o MVP do Vesta, um app de foco com timer, chama visual, tarefas e acompanhamento de consistencia.

## Scripts

```powershell
npm.cmd run dev
npm.cmd run build
npm.cmd run lint
npm.cmd run preview
```

No PowerShell, use `npm.cmd` caso a politica de execucao bloqueie o wrapper `npm.ps1`.

## Stack

- React
- TypeScript
- Vite
- CSS Modules
- lucide-react
